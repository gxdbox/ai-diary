## AI日记App UI设计方案

### 页面架构和导航流程

```
启动页 (Splash)
    ↓
首页 (Home) - Tab导航
    ├─ 录音页 (Record) - 主要入口
    ├─ 日记列表 (Timeline)
    ├─ 搜索页 (Search)
    └─ 分析页 (Analytics)
        ↓
日记详情页 (Detail)
    ├─ 编辑模式
    └─ 查看模式
```

```
请为AI智能日记App设计一个启动页面，要求：

风格：现代简约、温暖治愈

- 色彩柔和，不刺眼
```

## **页面2：录音页 (主页面)**

```
请为AI智能日记App设计录音页面，这是用户最常用的核心页面：

设备：iPhone 14 Pro (393x852pt)
场景：用户在上班路上边走边口述日记

页面布局：

【顶部区域】
- 状态栏：显示时间、信号
- 导航栏：
  - 左侧：返回按钮
  - 中央：今天的日期（如"3月18日 星期一"）
  - 右侧：设置图标

【中央区域 - 录音状态显示】
- 大圆形录音按钮（直径120pt）
  - 未录音状态：白色圆形，中间有麦克风图标
  - 录音中状态：红色脉动动画，显示音波
  - 暂停状态：橙色，显示暂停图标
- 录音时长显示（在按钮下方，如"00:02:35"）
- 音量可视化：环绕按钮的动态波形

【实时转写区域】
- 占据屏幕下半部分
- 白色卡片背景，圆角设计
- 标题："实时转写"（左上角）
- 转写文本区域：
  - 滚动显示转写内容
  - 最新的文字高亮显示
  - 支持自动滚动到最新内容
- 字数统计（右上角，如"已记录256字"）

【底部操作区】
- 三个按钮横向排列：
  - 左：清空按钮（垃圾桶图标）
  - 中：暂停/继续按钮
  - 右：完成按钮（对勾图标，主色调）

【交互状态】
1. 初始状态：大按钮居中，提示"点击开始录音"
2. 录音中：按钮变红，波形动画，实时显示文字
3. 暂停状态：按钮变橙，文字停止更新
4. 完成后：跳转到预览页面

设计要点：
- 大按钮设计，方便走路时单手操作
- 实时反馈，让用户知道正在录音
- 文字清晰可读，即使在户外强光下
- 配色温暖，减少焦虑感
```

## **页面3：日记预览和编辑页**

```
请设计日记预览和编辑页面，用户录音完成后会看到这个页面：

设备：iPhone 14 Pro (393x852pt)

页面布局：

【顶部导航】
- 左侧：返回按钮"< 重新录制"
- 右侧：保存按钮"保存"（主色调）

【内容区域】
分为两个Tab切换：

Tab 1: 原始文本
- 标签："原始转写"
- 显示语音识别的原始文本
- 浅灰色背景
- 包含"嗯、啊"等口语词
- 右上角显示"查看AI优化版本"按钮

Tab 2: AI优化文本（默认显示）
- 标签："AI优化"
- 显示清洗后的文本
- 白色背景
- 流畅的书面语
- 支持手动编辑

【AI分析卡片区域】
在文本下方显示三个分析卡片：

卡片1：情绪分析
- 图标：表情符号
- 显示识别的情绪（如"😊 平静愉悦"）
- 情绪强度条（1-10分）

卡片2：主题标签
- 图标：标签
- 自动生成的标签（如"#工作 #通勤 #思考"）
- 可点击编辑

卡片3：关键事件
- 图标：星标
- 提取的重要事件（如"决定周末去爬山"）

【底部操作】
- 大按钮："保存日记"
- 下方小字："保存后可在时间轴查看"

设计要点：
- 对比展示原文和优化文本，让用户看到AI的价值
- 分析结果可视化，增加趣味性
- 支持编辑，给用户控制权
```

## **页面4：日记时间轴（列表页）**

```
请设计日记时间轴列表页面：

设备：iPhone 14 Pro (393x852pt)

页面布局：

【顶部】
- 大标题："我的日记"
- 统计信息：
  - 左侧：总日记数（如"已记录365篇"）
  - 右侧：连续天数（如"🔥 连续30天"）
- 筛选按钮：
  - 按情绪筛选
  - 按主题筛选
  - 按时间筛选

【时间轴列表】
采用卡片式设计，每个日记卡片包含：

卡片布局：
- 左侧：日期标签
  - 月份日期（大字号，如"3月18日"）
  - 星期（小字号，如"周一"）
- 右侧：内容区域
  - 情绪图标（右上角，如😊）
  - 日记摘要（2-3行文字预览）
  - 主题标签（如"#工作 #健康"）
  - 底部元数据：
    - 字数（如"256字"）
    - 录音时长（如"2分35秒"）

卡片样式：
- 白色背景，轻微阴影
- 圆角12pt
- 卡片间距16pt
- 左侧日期标签用主题色高亮

【特殊标记】
- 重要日记：左侧有星标标记
- 今天的日记：卡片有渐变边框

【底部导航栏】
Tab Bar包含4个选项：
1. 时间轴（当前页，高亮）
2. 录音（中央大按钮，突出显示）
3. 搜索
4. 分析

【空状态】
如果没有日记：
- 中央显示插画（一个人在思考）
- 文字："开始记录你的第一篇日记吧"
- 大按钮："开始录音"

设计要点：
- 时间轴清晰，方便快速浏览
- 卡片设计信息密度适中
- 情绪图标增加视觉趣味
- 支持下拉刷新
```

## 整体设计规范

**配色方案：**
- 主色：温暖紫色 #8B7EC8
- 辅助色：浅蓝色 #6BB6D6
- 背景色：浅灰 #F5F5F7
- 文字色：深灰 #1D1D1F
- 成功色：绿色 #34C759
- 警告色：橙色 #FF9500
- 错误色：红色 #FF3B30

**字体规范：**
- 大标题：28pt，粗体
- 标题：20pt，中粗
- 正文：16pt，常规
- 辅助文字：14pt，常规
- 小字：12pt，常规

**圆角规范：**
- 卡片：12pt
- 按钮：8pt
- 输入框：8pt

**间距规范：**
- 页面边距：16pt
- 卡片间距：16pt
- 元素间距：8pt
