## 五、与松果日记集成方案

### 方案设计

1. **解耦原则**：AI伴侣独立服务，松果日记通过API调用
2. **触发时机**：用户写完日记 → 自动请求AI回应
3. **数据对接**：日记内容作为prompt上下文

### API对接示例

```python
# 松果日记创建日记后调用AI伴侣
diary_response = requests.post(
    "https://ai-companion.51pic.xyz/api/diary/respond",
    json={"diary_id": diary_id}
)
ai_message = diary_response.json()["ai_response"]
```
