## 二、技术架构

### 当前架构（v1.0）

```
┌─────────────────────────────────────────────┐
│              FastAPI 后端 (v1.0)              │
├─────────────────────────────────────────────┤
│  聊天引擎 (DeepSeek API)                     │
│  - 对话记忆（滑动窗口 5轮）                    │
│  - 情绪匹配回应                              │
│  - Prompt版本管理                            │
├─────────────────────────────────────────────┤
│  API 路由                                    │
│  - POST /api/chat (带记忆)                   │
│  - POST /api/diary/respond (情绪匹配)        │
│  - POST /api/feedback (用户反馈)             │
│  - GET /api/feedback/stats (统计)            │
├─────────────────────────────────────────────┤
│  SQLite 数据库                               │
│  - diaries (日记数据)                        │
│  - conversations (对话历史)                  │
│  - feedbacks (用户反馈)                      │
└─────────────────────────────────────────────┘
```
