## 十一、重点测试场景说明

### 🔴 核心路径（必须全部通过）

1. **语音录制 → AI清洗 → 查看详情** - 完整日记创建流程
2. **AI问答 → 反馈 → 再问一次** - 搜索核心功能
3. **筛选 → 查看 → 清除筛选** - 时间线基础操作

### 🟡 新功能验证（重点）

1. **隐性记忆表达** - 验证 AI 不直接复述用户历史
2. **反馈学习机制** - 验证反馈影响后续检索质量
3. **上下文连续性** - 连续多轮对话测试

### 🟢 边界场景

1. 无网络/弱网络场景
2. 大量数据场景（100+条日记）
3. 快速操作/重复点击
